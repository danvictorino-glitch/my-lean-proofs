# My Lean Proofs

This repository contains Lean 4 source files for the AEO project.

Prerequisites:
- Install Git: https://git-scm.com/downloads
- Install Lean 4 and Lake following Mathlib4 instructions: https://leanprover-community.github.io/mathlib4_docs/

Quick local commands to initialize and push:

```bash
git --version
git init
git add .
git commit -m "My first Lean 4 proof"
git branch -M main
git remote add origin https://github.com/your-username/my-lean-proofs.git
git push -u origin main
```

If `git` is not installed on your machine, install it first and then run the commands above from the repository root (`c:\Users\LOY\.vscode\cli`).
